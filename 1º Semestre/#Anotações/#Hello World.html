1. Descrever a origem, propósito e características da linguagem HTML:

HyperText Markup Language, criada no início dos anos 1990 por Tim Berners-Lee, é a linguagem usada para criar a estrutura de páginas web. Sua principal função é definir a estrutura de uma página através de tags, que são interpretadas pelos navegadores para exibir conteúdo como texto, imagens, links e outros. As especificações e padrões da HTML são geridos pelo World Wide Web Consortium. A versão atual da linguagem é a HTML5, que introduziu novos recursos como vídeos, gráficos e maior suporte a dispositivos móveis​.


2. Descreva o software necessário para se criar e testar páginas HTML:

Para criar páginas HTML, você precisa de um editor de texto (como Visual Studio Code, Sublime Text ou até o Bloco de Notas) e um navegador web (como Chrome, Firefox ou Edge) para visualizar e testar o código. Ferramentas como o VS Code também oferecem extensões para facilitar a edição e a depuração de código HTML.


3. Descreva a finalidade das seções e de uma página HTML:

  • Seção <head>: Contém informações de configuração e metadados da página, como o título que aparece na aba do navegador, links para arquivos de estilo (CSS) e scripts (JavaScript), e metatags que ajudam na indexação e responsividade. Essa seção é processada pelo navegador antes da exibição do conteúdo da página​.
  • Seção <body>: Contém todo o conteúdo visível da página, como textos, imagens, links e outros. Esta é a área onde o conteúdo real da página é exibido no navegador​. Tudo o que o usuário vê e interage na página está dentro dessa seção.
